# 维吾尔语打字练习系统 (Simplified)

一个专注于维吾尔语（Uyghur）打字练习的在线系统。

## 📋 功能特性

- ✅ 专门为维吾尔语设计的打字练习
- ✅ 实时打字速度（WPM）和准确率统计
- ✅ 维吾尔语软键盘显示/隐藏切换
- ✅ 响应式设计，适配不同屏幕
- ✅ 简洁清爽的界面

## 🚀 快速开始

### 前端启动

1. 进入前端目录
```bash
cd frontend
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```
